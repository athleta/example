# その他１問目
・Scratchならではの処理である「メッセージ」を利用してみる。<br>
「メッセージ」では複数のスプライトを使用するため、あらかじめ「スプライトを選ぶ」からスプライトをダウンロードしておく。（例題では「Earth」を使用）
スプライトが同じ座標にあると見辛いので、あらかじめ片方のスプライトを画面上部や下部に移動させておく。<br>
画面右側のスプライトから、それぞれのスプライトを選ぶことでスプライトごとにプログラミングを行うことができる。<br>
「（メッセージ）を送る」処理を猫のスプライトで、「（メッセージ）を受け取った時」処理を地球のスプライトで使う。<br>
猫のスプライトが１５度回った後にメッセージを送り、メッセージを受け取ったスプライトも１５度回るプログラムをそれぞれ作成する。<br>

# その他２問目
・繰り返し処理をメッセージとともに使用することで、メッセージを複数回送る<br>
質問の答えを繰り返す回数として使用したい場合、「（）と聞いて待つ」を使用した後、「（）回繰り返す」の（）内に「（答え）」ブロックを入れる。<br>
「（メッセージ）を送る」を繰り返し行う処理として設定することで、質問は猫のスプライトが行い、メッセージを送られたスプライトがメッセージを受け取った後の処理を入力された答えの回数分行う<br>
「(メッセージ）を送る」処理の後に「（）秒待つ」を入れていないと、メッセージを送った後すぐにメッセージを送るため、答えの値が小さいとすぐに処理が終わってしまい、「１５度回す」がすぐに終わってしまうため注意。<br>
答えを入力するときは半角数字でなければプログラムが正常に動かないので注意。<br>

# その他３問目
・条件を満たした場合のみメッセージを送るプログラムを作成する。
ループ中に条件を満たした場合の処理の考え方は、条件分岐第７問で使用した考え方を使用する。<br>
「10歩動かし（もし端に着いたら、跳ね返る）、もしも変数が１０以下ならメッセージを送り、変数を1ずつ変える」処理をずっとループさせる。<br>
