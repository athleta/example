# 繰り返し５問目
・繰り返しを使用することで、スプライトを使用してアニメーションを作る。<br>
コスチューム差分はデフォルトで用意されているので、画面左上の「コスチューム」から確認する。<br>
コスチュームを切り替えた後、「()秒待つ」を入れなければ、コスチュームを切り替えた後の感覚がないため、ずっと同じコスチュームに見えてしまうため注意。<br>

# 繰り返し６問目
・ずっと繰り返すのではなく、回数を指定した繰り返しを使用することで繰り返しを用いたプログラムに幅を持たせる。<br>
繰り返し問題３の「ずっと〜する」ブロックを「（）回繰り返す」に変更することで、視覚を用いて違いを理解する。<br>

# 繰り返し７問目
・タイマーを用いて、回数ではなく時間を指定したループを作成してみる。<br>
「タイマー」とはScratch上でカウントされている内部時計で、プログラムを動かした時、タイマーのカウントも始まる。<br>
最初に「タイマーを０にする」ブロックを使用していないと、何らかのプログラムでタイマーが起動していた時に処理結果が変わってしまうので忘れないようにする。<br>
ループを繰り返す条件は「タイマー > 5」で、色の効果は20ずつ変えていく。<br>

# 繰り返し８問目
・様々な条件下で繰り返し処理を行うことで、回数や時間以外を指定して繰り返し処理への理解をさらに深める。<br>
ループを繰り返す条件は「x座標 = ５０」になるまで。<br>
x座標およびy座標は10ずつ変化している。<br>

# 繰り返し９問目
・繰り返しの中に別の繰り返しを組見合わせることで、複数の繰り返し処理は共存できるものであることを理解する。<br>
「10歩動く」と「端に着いたら跳ね返る」を3回繰り返したものと、「15度回す」を4回繰り返したものを「ずっと〜する」を用いて繰り返す。<br>

# 繰り返し１０問目
・繰り返しの中に別の繰り返しを組見合わせることで、複数の繰り返し処理は共存できるものであることを理解する。<br>
「10歩動く」と「端に着いたら跳ね返る」を５回繰り返したものと、「y座標を20ずつ変える」「y座標を０にする」を2回繰り返したものを「ずっと〜する」を用いて繰り返す。<br>
「y座標を20ずつ変える」と「y座標を０にする」の間に「（）秒待つ」を入れなければ、y座標が２０変わったことを確認できないので注意。<br>

# 繰り返し１１問目
・繰り返しの中に別の繰り返しを組見合わせることで、複数の繰り返し処理は共存できるものであることを理解する。<br>
「10歩動かした後、音を鳴らす処理を2回繰り返す」処理を3回繰り返す。<br>
繰り返し問題第２問のように、音を鳴らした後には「()秒待つ」処理を入れる。
